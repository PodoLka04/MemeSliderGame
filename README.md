# Пятнашки с Тунг-Тунг Сахуром!

Проект создан для публикации на платформе **Yandex Games** с целью коммерции.

**Платформа:** WebGL (для ПК и мобильных устройств)

## 🎮 О проекте

**Пятнашки с Тунг-Тунг Сахуром!** — это головоломка с героями самых эпичных итальянских мемов. Каждый уровень — новый харизматичный персонаж и их обаятельные приключения. После победы тебя ждёт забавный отрывок с историей от любимых героев.

## 🎯 Основные механики

- **Головоломка:** Собирай пятнашки, перемещая плитки на пустое поле.
- **Персонажи:** Каждый уровень — новый герой итальянских мемов.
- **Награды:** За победу открываются смешные видео с персонажами.
- **Прогресс:** Начинай с простых уровней и прокачивай навыки для сложных головоломок.
- **Галерея:** Коллекция наград за пройденные уровни.

## 🚀 Сборка и запуск

1. Скачайте репозиторий.
2. Откройте проект в Unity (версию смотрите в `MineSlider/ProjectSettings/ProjectVersion.txt`).
3. Для запуска в редакторе откройте сцену `MineSlider/Assets/Scenes/MainScene.unity` и нажмите `Play`.
4. Для публикации на Yandex Games соберите проект в **WebGL** через `File → Build Settings`.
5. Загрузите сборку в личном кабинете разработчика Yandex Games.

# Fifteen Puzzle with Tung-Tung Sahur!

The project was created for the **Yandex Games** platform in profit.

**Platform:** WebGL (for PC and mobile devices)

## 🎮 About the project

**Fifteen Puzzle with Tung-Tung Sahur!** is a puzzle game featuring heroes from the most epic Italian memes. Each level introduces a new charismatic character and their charming adventures. After winning, you'll be rewarded with a funny clip featuring your favorite heroes.

## 🎯 Core mechanics

- **Puzzle:** Solve fifteen puzzles by moving tiles into the empty space.
- **Characters:** Each level features a new hero from Italian memes.
- **Rewards:** Unlock funny videos of the characters after winning.
- **Progression:** Start with easy levels and improve your skills for harder puzzles.
- **Gallery:** Collect rewards for completed levels.
